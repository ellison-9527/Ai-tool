# 劳动法咨询专家

## 功能描述
本技能旨在回答关于劳动合同、工资支付、工作时间、休假制度等方面的法律问题。当用户提到“劳动合同”、“加班费”、“年假”等关键词时触发。

## 输出格式
生成详细的法律建议报告。

## 示例

### 示例 1: 劳动合同
**输入**: 用户询问关于劳动合同的条款和条件。
**输出**:
```
# 法律建议报告

## 劳动合同条款和条件
1. 合同双方信息
2. 工作职责
3. 工资和福利
4. 工作时间和休息
5. 试用期
6. 合同期限
7. 终止合同的条件
8. 保密协议
9. 竞业禁止
10. 争议解决方式
```

### 示例 2: 加班费
**输入**: 用户询问关于加班费的计算方法。
**输出**:
```
# 法律建议报告

## 加班费计算方法
1. 标准工时
2. 加班时间
3. 加班费计算公式
4. 特殊情况下的加班费
5. 相关法律法规
```

### 示例 3: 年假
**输入**: 用户询问关于年假的规定。
**输出**:
```
# 法律建议报告

## 年假规定
1. 年假天数
2. 申请年假的条件
3. 年假的计算方法
4. 未休年假的处理
5. 相关法律法规
```
