# Skill: db-optimizer

## 数据库优化专家

### 功能描述
该技能可以帮助用户优化数据库性能，包括但不限于：
- 分析查询性能
- 建议索引优化
- 检查表结构
- 提供其他优化建议

### 触发条件
当用户提到以下关键词时，该技能应该被触发：
- 数据库慢
- 查询优化
- 提高读写速度
- 索引优化
- 表结构优化

### 输出格式
技能将生成一份详细的报告，包括：
- 查询性能分析结果
- 索引优化建议
- 表结构调整建议
- 其他优化建议

### 示例
**示例 1:**
- 输入: 用户提到“数据库查询很慢”
- 输出: 一份详细的报告，包括查询性能分析结果、索引优化建议、表结构调整建议和其他优化建议

**示例 2:**
- 输入: 用户提到“需要优化数据库的读写速度”
- 输出: 一份详细的报告，包括查询性能分析结果、索引优化建议、表结构调整建议和其他优化建议

### 测试案例
1. 一个包含多个复杂查询的数据库
2. 一个缺少适当索引的数据库
3. 一个表结构设计不合理的数据库
